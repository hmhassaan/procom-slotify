# **App Name**: Course Extractor

## Core Features:

- Excel Upload: Allow users to upload an Excel file containing the timetable data.
- Data Extraction: Extract course information and timings from the first five sheets (Monday to Friday) of the Excel file, reading times from cells B3-J3 and course data from B5-J72.
- Course Selection: Make the extracted courses selectable for further actions.
- User Creation: Allow users to add their name and select courses they are enrolled in.
- Timetable Analysis: Analyze user schedules and display time slots when a specific user does not have class.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke a sense of clarity and organization, suitable for an educational tool.
- Background color: Very light blue (#F0F8FF), close to white, for a clean and unobtrusive background that keeps the focus on the extracted timetable data.
- Accent color: Muted teal (#70A1AF) to complement the blue primary, while creating some visual interest and a professional feel.
- Font: 'PT Sans', a humanist sans-serif font suitable for both headings and body, which combines modernity with approachability.
- Use simple and clear icons to represent different actions like uploading, user creation, and timetable analysis.
- Use a clear and tabular layout to present the extracted course information and timetable analysis.
